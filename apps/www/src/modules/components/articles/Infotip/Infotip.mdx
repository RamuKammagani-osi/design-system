## Examples

### UncontrolledInfotip

For a _controlled_ `Infotip` to function correctly, it requires event handlers to be passed in as props and focus/hover states to be managed. It is far more convenient to use the `UncontrolledInfotip` that does this for you.

```jsx
import { UncontrolledInfotip } from "@cwds/components";
```

<Card>
  <CardBody>
    <CardTitle className="mb-0">
      Simpler with UncontrolledTooltip?
      <UncontrolledInfotip id="my-uncontrolled-infotip" placement="right">
        <p className="mb-0">
          The <code>UncontrolledInfotip</code> is here to keep it simple. If you
          don't want to manage state, this is the way to go!
        </p>
      </UncontrolledInfotip>
    </CardTitle>
  </CardBody>
</Card>
