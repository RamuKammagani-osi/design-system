## Synopsis

Use a `Multi Select` to pick n from an option of values. For more docs [react-select](https://react-select.com/home#getting-started)

<DemoCard>
  <SimpleSelectExample options={TOPPINGS} />
</DemoCard>

```jsx
import { Select } from "@cwds/forms";
```

## Props

<!-- <PropTable docgen={Docgen.Select} /> -->

use `isMulti` prop and set it to `true` to transform the a simple `Select` to `Multi Select`

## Usage Guidelines

Use `Multi Select` when a large list of options are available and picking n values.
