### Textareas

If a <em>long-form</em> text input is desired, use `type="textarea"`:

<DemoCard>
  <Input type="textarea" placeholder="Another Placeholder..." rows="5" />
</DemoCard>

```jsx
<Input
  type="textarea"
  placeholder="Another Placeholder..."
  rows="5"
  resize="vertical"
/>
```
