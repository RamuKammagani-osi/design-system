## Synopsis

Please see also the <Link to="/components/forms/input#textareas">Textarea section in Input Article</Link>.

Use a `Text Area` to enter a multi-line input field to collect larger, free-form text from a user.

<DemoCard>
  <TextArea
    aria-label="Feedback"
    label="Feedback"
    maxLength="100"
    placeholder="Tell us your Feedback"
  />
</DemoCard>

```jsx
import { TextArea } from '@cwds/components'
```

## Usage Guidelines

Use `TextArea` when a user inputs large list of characters.

Use `TextAreaCounter` when a user inputs large list of charcacters and see the remaining characters.

```jsx

```

<ReferenceCard>
  This is a controlled component. You need to manage state. It uses Formik to
  manage state.
</ReferenceCard>

<Card>
  <CardBody>
    <Formik
      initialValues={{ message: '' }}
      render={props => {
        return (
          <TextAreaCounter
            aria-label="Feedback"
            label="Feedback"
            maxLength="50"
            onChange={e => props.setFieldValue('message', e.target.value)}
            placeholder="Tell us your Feedback"
            rows="5"
            value={props.values.message}
          />
        )
      }}
    />
  </CardBody>
</Card>

```jsx
<TextAreaCounter
```
