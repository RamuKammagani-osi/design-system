## Synopsis

Use an `Input Mask` in to capture phone numbers, dates, social security input in forms.

<DemoCard>
  <MaskedInput
    mask={[
      "(",
      /[1-9]/,
      /\d/,
      /\d/,
      ")",
      " ",
      /\d/,
      /\d/,
      /\d/,
      "-",
      /\d/,
      /\d/,
      /\d/,
      /\d/
    ]}
    render={(ref, props) => <Input innerRef={ref} {...props} />}
    showMask
  />
</DemoCard>

```jsx
import { MaskedInput } from "@cwds/forms";
```
