## Synopsis

Use an `Input Mask` in to capture phone numbers, dates, social security input in forms.

  <InputMask mask={myMask} showMask />
</DemoCard>


### Common Types

For date types, use `MASKS.DATE`:

<Card>
  <CardBody>
    <InputMask mask={MASKS.DATE} placeholder="12/07/1941" />
  </CardBody>
</Card>

```jsx
<InputMask mask={MASKS.DATE} placeholder="12/07/1941" />
```

For telephone types, use `MASKS.PHONE`:

<Card>
  <CardBody>
    <InputMask mask={MASKS.PHONE} placeholder="(555) 555-5555" />
  </CardBody>
</Card>

```jsx
<InputMask mask={MASKS.PHONE} placeholder="(555) 555-5555" />
```
