### PrimitiveButtons

<ReferenceCard>
  The <code>Button</code> component exported from <code>@cwds/components</code>{" "}
  is an abstraction on top of the <code>Button</code> from{" "}
  <code>reactstrap</code>. We've done this promote consistency of button usage
  in CWDS service modules. If you <em>must</em> drop down into{" "}
  <code>reactstrap</code>'s <code>Button</code>, it is exported as{" "}
  <code>PrimitiveButton</code>.
</ReferenceCard>

Buttons take a color prop which should be one of the thematic color values:

<div className="my-3">
  <PrimitiveButton color="primary">primary</PrimitiveButton>{" "}
  <PrimitiveButton color="secondary">secondary</PrimitiveButton>
</div>

```jsx
<PrimitiveButton color="primary">primary</PrimitiveButton>
<PrimitiveButton>secondary</PrimitiveButton>
```
